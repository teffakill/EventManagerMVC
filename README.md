# Sistema de Gestión de Eventos MVC

Este proyecto es una aplicación desarrollada en Java utilizando el patrón MVC (Modelo - Vista - Controlador) y componentes Swing.

## Funcionalidades

* Registrar eventos
* Listar eventos
* Eliminar eventos
* Registrar invitados

## Estructura del Proyecto

* Model: Clases como Evento e Invitado
* View: Interfaces gráficas con Swing
* Controller: Lógica del sistema

## Tecnologías

* Java
* Swing
* GitHub
